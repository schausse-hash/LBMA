📋 Contrat d'équipe — Softball Québec

Génère les contrats d'équipe officiels pré-remplis depuis la base de données

⚙️ Paramètres

👥 Joueurs

# Section Nom Prénom Naissance Adresse Ville Code postal Tél. Courriel

ℹ️ Informations

Cet outil génère un fichier Excel (.xlsx) reproduisant le formulaire officiel Contrat d'équipe de Softball Québec, avec 2 onglets (Français / English). Les données sont pré-remplies depuis la table joueurs_liste et equipes_saison.

Sections remplies automatiquement :
• JOUEURS (max 17) — rôles joueur et lanceur_a
• ENTRAÎNEURS (max 5) — rôles coach, coachlanceur_a, coachadmin
• GÉRANT — à remplir manuellement dans le Excel

Données utilisées : nom, naissance, adresse, ville, code_postal, telephone, courriel
Date limite Softball Québec : 15 juin de chaque saison
